# BEdita SDK JavaScript

BEdita SDK for JavaScript.

## Dev

### Requirements

* [RNA](https://gitlab.com/chialab/rna-cli) (>= 0.19.0)
* [Node](https://nodejs.org) (>= 6.0.0)

### Build

```sh
rna build
```

### Demo

```sh
rna start
```
