# BEdita SDK JavaScript

BEdita SDK for JavaScript.

[![npm](https://img.shields.io/npm/v/bedita-sdk-js.svg)](https://www.npmjs.com/package/bedita-sdk-js)

## Install

```sh
npm install bedita-sdk-js
```

## Dev

### Requirements

* [Node](https://nodejs.org) (>= 6.0.0)

### Build

```sh
npm install
npm run build
```
